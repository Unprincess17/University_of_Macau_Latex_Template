# 用于澳门城市大学课程作业的Latex模板



## 说明

* `main.pdf` 中可以预览模板效果
* `main.tex` 中进行正文编写
* `CityUhomework.sty`中编辑标题、页眉、学院、专业、作者等信息
* `reference.bib`中是参考文献列表
* `figures` 此文件夹存放所需的图片
